# AI Crop Disease Detection

This project uses a deep learning model to detect plant diseases from leaf images.

## Features
- Upload plant leaf image
- Predict disease using trained model
- Visual confidence graph
- Disease severity visualization

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- Plotly


## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py


----------------
## Dataset
PlantVillage dataset used for training.
